## *목표*
* 1일 계획량을 목표로 강의를 수강하고 변경된 내역을 commit 한다.
* 강사님 코드만 사용하지 않고 다양하게 학습한다.
* 사이드 프로젝트 혹은 현재 업무에 적용할 수 있는 방안을 생각한다.

## *사용법*
* `git log` 를 통해 commit hash 확인
* `git checkout {hash_value}`
* main branch 원복: `git switch -`
